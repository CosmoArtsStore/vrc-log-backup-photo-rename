STELLA RECORD

VRChat ライフをより豊かに、より確実に。

Polaris

常駐型自動バックアップエンジン

  • VRChat のセッション終了を検知し、output_log.txt を自動的にアーカイブへ同期します。
  • 差分同期アルゴリズムにより、ネットワークやストレージへの負荷を最小限に抑えます。
  • バックアップ先: %LOCALAPPDATA%\CosmoArtsStore\STELLARECORD\app\Polaris\archive

Planetarium

ログ解析・データベース同期 Engine

  • Polaris が収集した生ログを解析し、ワールド訪問履歴をデータベース化します。
  • Danger Zone: データの不整合やリセットが必要な場合、DB初期化や強制再同期が可能です。
  • DB照会: SQLite データベースの全テーブルを直接ブラウズできる詳細ビューを搭載しています。

Pleiades Alpheratz

VR写真・メタデータマネージャー

  • StellaRecord 連携: アプリ上の「StellaRecord に接続」ボタンを押すだけで、自動的にランチャーへ登録されます。
  • VRChat で撮影した写真と、その瞬間のワールド情報を自動的に紐付けます。
  • メモ機能を使用して、思い出をテキストとして残すことが可能です。

⚠️ セキュリティ・重要事項

管理者権限について: ログファイルの読み取りやアプリの自動起動設定のため、初回実行時や設定変更時に管理者権限が必要になる場合があります。

システム概要

STELLA RECORD は、独立した複数のモジュールが連携して動作するマイクロサービス・アーキテクチャを採用しています。すべてのアプリは同一のデザインシステムで統一されており、シームレスな操作感を提供します。
