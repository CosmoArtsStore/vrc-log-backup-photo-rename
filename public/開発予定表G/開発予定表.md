## Alpheratz 同一ワールド 同一衣装判定機能を新規追加作成したい 画面には簡易的に結果出すだけでいい まずアルゴリズム作成とDbカラム追加からお願い
### 簡略化した構成

```
元画像
  ├── 背景マスク → CLIP埋め込み → world_emb
  └── 前景マスク → CLIP埋め込み → avatar_emb

検索モード
  ├── 「ワールドで探す」→ world_emb だけでコサイン類似度
  └── 「衣装で探す」  → avatar_emb だけでコサイン類似度
```

**2つのベクトルを独立して使うだけ**。組み合わせスコアの計算が不要になります。

---

### 削れるもの

- ~~`world_score × avatar_score` の複合判定~~ → 不要
- ~~「両方一致」クラスタ~~ → 不要
- ~~閾値の掛け合わせチューニング~~ → 各モードで1つの閾値だけ調整すればOK

```python
# 保存（インポート時に1回）
photo = {

    "avatar_emb": get_embedding(fg_image),  # 前景CLIP
}

# 検索（ワールドで探す）
results = sorted(photos, key=lambda p:
    cosine_similarity(query.world_emb, p.world_emb), reverse=True)

# 検索（衣装で探す）
results = sorted(photos, key=lambda p:
    cosine_similarity(query.avatar_emb, p.avatar_emb), reverse=True)
```
