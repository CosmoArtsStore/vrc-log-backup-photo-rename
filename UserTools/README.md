# UserTools

UserTools は、ユーザー主導で蓄積した作業メモや比較画像、公開対象に含めない補助資料を整理するためのフォルダです。

## 何を置く場所か

- 課題メモ
- 作業途中のスクリーンショット
- 公開前の確認用テキスト
- TODO や lessons などの内部運用メモ

## フォルダの使い分け

- `notes/`: 作業メモ、課題メモ
- `images/`: 比較画像、確認用キャプチャ
- `tasks/`: 個人用 TODO や lessons

## 向いていないもの

- 本番アプリの本体コード
- 配布用アイコンや公開ドキュメント
- 再生成が必要なビルド成果物
