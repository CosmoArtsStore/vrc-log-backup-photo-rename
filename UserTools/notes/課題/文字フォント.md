# STELLARECORDエコシステム フォント設計指針

## 採用フォント

### 日本語
**Noto Sans JP**（Google Fonts / OFL / 無料）
- 理由：癖がなく中立・日本語の視認性が最も安定・BIZ UDより軽量・Tauriで同梱可能

### 英字・数字（UI全般）
**Noto Sans JP** で統一（英字フォントは別途指定しない）
- 理由：日英混在テキストでフォント切り替えが発生せず、一貫した印象を保てる

### 数字・ログ・パス・コード表示専用
**JetBrains Mono**（OFL / 無料）
- 理由：等幅・数字のゼロとOの区別が明確・ログ表示・ファイルパスで可読性が高い

## font-weight 使用ルール

| 用途 | weight | 適用例 |
|---|---|---|
| ページタイトル・モーダルタイトル | **700** | 「設定」「Planetarium Control」 |
| セクションヘッダー | **600** | 「参照フォルダ 1st」「タグマスタ」 |
| ボタンラベル | **600** | 「変更」「追加」「インポート開始」 |
| 本文・説明文 | **400** | 「Windows ログイン時に〜」 |
| サブテキスト・補足・空状態 | **400** + opacity 0.5〜0.6 | 「タグマスタはまだありません。」 |
| ログ・パス表示（JetBrains Mono） | **400** | ファイルパス・DB値 |

**原則：400と700の2極で階層を作る。500・600はボタンのみ。300は使わない。**

---

## font-size スケール

```
タイトル（モーダル・ページ）  : 20px / weight 700
セクションラベル              : 12px / weight 600 / letter-spacing 0.03em
本文・入力値・トグルラベル    : 13px / weight 400
ボタンラベル                  : 13〜14px / weight 600
サブ・空状態テキスト          : 12px / weight 400 / opacity 0.55
ログ・パス（JetBrains Mono）  : 12px / weight 400
```

## アプリ別適用方針

| アプリ | フォント | 備考 |
|---|---|---|
| **Alpheratz** | Noto Sans JP + JetBrains Mono | ライト・ダーク両対応。写真が主役なので本文は細め（400）を基本とする |
| **STELLARecord** | Noto Sans JP + JetBrains Mono | ログ・DB値の表示が多いのでJetBrains Monoの出番が多い |
| **Polaris** | Noto Sans JP | UIがほぼないため最小限の適用 |
| **Mira** | Noto Sans JP | 日記・感情的コンテンツ。本文は400・余白を広めに取る |

## CSS変数定義（共通トークン）

```css
:root {
  /* フォントファミリー */
  --font-base: 'Noto Sans JP', sans-serif;
  --font-mono: 'JetBrains Mono', 'Courier New', monospace;

  /* フォントウェイト */
  --fw-regular:   400;
  --fw-semibold:  600;
  --fw-bold:      700;

  /* フォントサイズ */
  --fs-title:     20px;
  --fs-section:   12px;
  --fs-body:      13px;
  --fs-button:    13px;
  --fs-sub:       12px;
  --fs-mono:      12px;

  /* letter-spacing */
  --ls-section:   0.03em;
  --ls-body:      0;
}
```

## Tauri（Rust）での同梱方法

```
src-tauri/
  assets/
    fonts/
      NotoSansJP-Regular.woff2
      NotoSansJP-SemiBold.woff2
      NotoSansJP-Bold.woff2
      JetBrainsMono-Regular.woff2
```

```css
@font-face {
  font-family: 'Noto Sans JP';
  src: url('/assets/fonts/NotoSansJP-Regular.woff2') format('woff2');
  font-weight: 400;
  font-display: block;
}
@font-face {
  font-family: 'Noto Sans JP';
  src: url('/assets/fonts/NotoSansJP-SemiBold.woff2') format('woff2');
  font-weight: 600;
  font-display: block;
}
@font-face {
  font-family: 'Noto Sans JP';
  src: url('/assets/fonts/NotoSansJP-Bold.woff2') format('woff2');
  font-weight: 700;
  font-display: block;
}
@font-face {
  font-family: 'JetBrains Mono';
  src: url('/assets/fonts/JetBrainsMono-Regular.woff2') format('woff2');
  font-weight: 400;
  font-display: block;
}
```

- `font-display: block` を指定することでフォント未ロード時のちらつき（FOUT）を防ぐ
- woff2のみで十分（Tauriは最新Chromiumベースのため）

---

## 採用理由まとめ（1行）

> Noto Sans JPは日英混在・日本語UIで最も破綻しない選択肢であり、JetBrains Monoはログ・数値表示の視認性を補完する。font-weightは400と700の2極で階層を作る。
