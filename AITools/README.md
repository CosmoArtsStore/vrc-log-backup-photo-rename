# AITools

AITools は、AI を使った調査や補助作業のためのフォルダです。  
本体アプリではなく、調査スクリプトや分析結果を整理して置くために使います。

## 何を置く場所か

- 調査用スクリプト
- 変換や抽出の補助ツール
- 分析メモ
- AI が作成した出力結果

## フォルダの使い分け

- `scripts/`: 実行するスクリプト
- `docs/`: 調査メモや手順書
- `output/`: 実行結果や抽出物

## 使うときの注意

- 新しいファイルをルート直下へ増やしすぎない
- 後で作り直せる出力は `output/` に寄せる
- スクリプト名は、見ただけで用途が分かる名前にする

## 向いていないもの

- 本番アプリの本体コード
- ユーザーが直接使う設定ファイル
- 消してはいけない手作業データ
