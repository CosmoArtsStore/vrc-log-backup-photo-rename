# Polaris

Polaris は、STELLAProject 内で他アプリのデータやログに関わる処理を担当するアプリです。  
ユーザーデータや他アプリの実ファイルに触れる可能性があるため、特に慎重な運用が必要です。

## 取り扱い注意

- 他アプリの動作を邪魔しないこと
- ユーザーデータを壊さないこと
- 停止したときの影響を説明できること

## 開発時の基本方針

- 難しい抽象化より、読んで追える実装を優先する
- ファイル操作やログコピーの条件は、理由が分かるようにコメントを残す
- 失敗したときは黙って無視せず、何を諦めて何を続けるかを分かるようにする

## 利用者向けメモ

Polaris は内部処理の比重が高いため、単体で直接操作する画面が少ない場合があります。  
他アプリと連携して動く前提のため、設定変更やファイル配置を行う前に手順書を確認してください。

## 開発者向け

- 変更前に、どのファイルへ触れるのかを文章で説明できる状態にしてください。
- 読み取り専用で済む処理と、書き込みが発生する処理を分けて考えてください。
- 安全性が説明できない変更は入れないでください。
